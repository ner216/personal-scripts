
# Bash Scripts

- `awake` : (Generated via ChatGPT) Prints a breakdown of time the system spent suspended and awake since last boot.
- `extract` : (Generated via Gemini) Extracts gresource binary files. (These files are used to compress GTK themes)
- `lsgit` : List or show status on all git repos on system.
- `config-ubuntu-vanilla.sh` : Remove Yaru theme and snap from Ubuntu.
